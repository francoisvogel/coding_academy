Guide

The experts' guide on how to boost your coding skills

1. Learning a programming language

The first and most important step to boost your coding skills is to learn a programming language.
The most obvious choice is probably Python, an interpreted high-level programming language with an object-oriented approach. It has a very simple and visual syntax (through indentation): a perfect choice for beginners! A good website to start training with would probably be HackerRank. However it's possible to start coding with other languages, Francois personally himself started with Processing.
Then you can move over to compiled languages like C/C++ which are very fast, especially in competitive programming contests. However these introduce new concepts, which is why - to practice - we would recommend Codeforces, a website that offers a set of problems, along with solutions. If you're interested in web development, a possibility would be to learn Javascript/Typescript, so that you can create dynamic webpages, eventually with a Node.js back-end.
You can easily code in-browser using the in-built IDE (Visual Studio Code).

2. Problem-solving

To improve your Data Structures & Algorithms skills, you need to learn techniques and solve some problems. In our recorded biweekly lectures, we try to cover as much topics as possible in a wide range of areas. You can always review them afterwards in our YouTube channel..
We provide a collection of high-quality, tailored problems. Each one of them comes up with its own editorial and even a sample solution. We have developed a difficulty rating system for our problems, ranging from 0 to 1000:
Difficulty Level Target
0-300 Easy These problems are targeted to newbies in programming.
300-500 Medium Problems in this range are for experienced coders.
500-800 Hard Problems in this range are for seasoned programmers who regularly train on competitive programming platforms.
800-1000 Extrahard This range of problems is aimed at red or future red coders and beyond.
The difficulty value can be found for every problem in a black circle.

3. Productive Development

Being a productive developer is essential. We recommend using an IDE such as Visual Studio Code (which is now available in-browser), Atom, PyCharm (the best IDE for python) or CLion (a great IDE for C/C++). It's then necessary to install tools such as linters, debuggers, live servers to make the development process faster and more efficient. Contact us if you have are experiencing any issues or have questions regarding this.
