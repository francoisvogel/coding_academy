1. Learning a programming language

The first and most important step to boost your coding skills is to learn a programming language.
The most obvious choice is probably Python, an interpreted high-level programming language with an object-oriented approach. It has a very simple and visual syntax (through indentation): a perfect choice for beginners! However it's possible to start coding with other languages, I personally started with Processing. A good website to begin with would probably be HackerRank
